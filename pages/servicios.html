Nuestra experiencia volcada en sugerencias

  1. Verificar que este conectado a la alimentacion tanto el router como el access point.

  2. Apagar el router y el access point cuando haya actividad electrica por tormenta.

  3. Reiniciar cada 7 días el router y access point.

Aclaración de Términos y Conceptos Utilizados

Router
* Dispositivo que permite interconectar computadoras que funcionan en el marco de una red.
Access Point
* Punto de acceso cuya finalidad es autenticar el acceso a la red inalámbrica
Inyector Poe
* Proporciona alimentación eléctrica a un dispositivo a través de un único cable de red Ethernet.
